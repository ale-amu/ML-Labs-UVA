# Machine Learning Labs — University of Virginia (DS 3021)

## Overview
A collection of hands-on machine learning labs completed as part of UVA's undergraduate Data Science curriculum. Each lab applies end-to-end ML workflows to real-world datasets, covering everything from data cleaning and exploratory analysis to model building, tuning, and evaluation.

---

## Labs

### Decision Tree Classification | Movie IMDB Score Prediction
- Built a decision tree classification pipeline to predict whether a movie would receive a high IMDB score
- Performed data cleaning, variable type correction, and feature engineering on a 5,000+ record movie dataset
- Implemented full Scikit-learn Pipeline with OrdinalEncoder and ColumnTransformer for preprocessing
- Tuned hyperparameters using GridSearchCV and RepeatedStratifiedKFold cross-validation
- Achieved final test AUC of 0.84 and Recall of 0.91 after iterative model improvement
- Evaluated using AUC, Recall, Balanced Accuracy, and Accuracy metrics

### Clustering Lab | NBA Player Salary & Performance Analysis
- Merged NBA performance and salary datasets to identify undervalued players using K-Means clustering
- Scaled features prior to clustering and selected optimal cluster count
- Visualized clusters and interpreted results to produce actionable scouting recommendations
- Communicated findings in a stakeholder-facing report for a non-technical audience

### Model Metrics Lab
- Evaluated classification model performance using confusion matrices, precision, recall, and AUC
- Compared multiple models and interpreted tradeoffs between metrics for real-world decision making
